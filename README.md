Project Name: Flipkart Sales Analysis Dashboard

Description:
This project is a data analysis and visualization dashboard created to analyze Flipkart sales data. It provides insights into total sales, units sold, brand performance, delivery time, discounts, returnable products, and category-wise trends using interactive charts and filters.

Objective:
To understand customer behavior, brand performance, city-wise sales distribution, and category trends to support better business decisions.

Features:

Total sales and units sold overview

Brand-wise rating analysis

Price distribution by brand

City-wise seller analysis

Category-wise warranty comparison

Returnable vs non-returnable sales

Average delivery days by brand

Interactive filters for city and category

Tools & Technologies Used:

Power BI

Excel / CSV dataset

Data cleaning & transformation

Data visualization techniques

Dashboard Insights:

Highest selling brands identified

Fashion and electronics category comparison

Delivery performance by brand

Returnable product percentage

City-wise seller contribution

How to Use:

Open the Power BI file

Use filters for city and category

Analyze charts and KPIs

Observe trends and patterns
